========
Settings
========

.. _admin_related_excludes:

**ADMIN_RELATED_EXCLUDES**
	A list or tuple of strings in ``'appname.model'`` format of models that should not be shown in any related items list.

.. _admin_related_item_limit:

**ADMIN_RELATED_ITEM_LIMIT**
	The maximum number of items that should be listed for each related model. **Default:** 5
