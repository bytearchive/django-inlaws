Django-InLaw’s v 0.1.1beta1¶

When you are editing an object, Django-InLaws shows a sidebar containing objects related to it in the database. It consists of a replacement template for the Admin’s change form and some template tags.

The only objects that are shown in the sidebar are those which have a registered Admin interface or are not explicitly excluded.
